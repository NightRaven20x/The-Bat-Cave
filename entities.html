Bat-Computer Entity Decoder

To safely output code to the Bat-Computer terminal without triggering HTML execution, we use the following character entities:

  • < &lt; used for Less Than (e.g., displaying tags like <div>)
  • > &gt; used for Greater Than
  • & &amp; used for Ampersand (Wayne & Enterprises)
  • " &quot; used for Double Quotes in strings
  • ' &apos; used for Single Quotes (Batman's gadgets)
  • © &copy; used for Copyright (© 2026 Wayne Tech)
  • ® &reg; used for Registered marks (Arkham Asylum®)
  • ™ &trade; used for Trademarks (Batmobile™)
  • [ ] &nbsp; used for Non-Breaking Spaces (stops text from wrapping)
  • £ &pound; used for Pound Sterling (Alfred's salary)
  • × &times; used for the Multiplication/Close symbol
